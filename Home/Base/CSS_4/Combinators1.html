Форматы файлов.

.DOC или .doc (от англ. document) - расширение имени файла, используемое для файлов, представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения простых текстовых файлов юез формирования, однако позже стало использоваться для двоичных форматов с разметкой.
MS Word

Portable document Format (от англ. PDF) - разработанный фирмой Adobe Systems с испольхованием ряда возможностей языка Postscript.
Документ PDF

ZIP - популярный формат сжатия и архивации файлов. Файл в этом формате обычно имеет расширение .zip и хранит в сжатом или несжатом виде один или несколько файлов, которое можно из него извлечь путём распаковки с помощью специальной программы.
